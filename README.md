# Github-Setup 

This assignment is to verify you can use Github correctly for all your future class lab/Homework assignments.

You need to complete the following tasks.

1. Create a simple HelloWorld.cpp program that prints "Hello World!" on the console. <b>(Make sure to name your .cpp file as HelloWorld.cpp)</b>
2. Run the program and verify the output is generated correctly.
3. Create a Github account and link your account to Github classroom ID provided for you.
4. Push your code.
